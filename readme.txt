Fast and easy integration of daily sudoku puzzles into your blogs. 

== Description ==

Major features in this version include: 

* The sudoku puzzles can be inserted everywhere: in your theme files, posts and pages.
* Seven different levels from simple to diabolic are available 
* Use pencilmarks by pressing ALT and Number key
* they are daily updated (upto 10 history entries) sudokus and can be used as daily sudoku 
* You can change appearance, size and visible features 
* printable sudoku for offline solving are available too
* supported languages are english, german, swedish, italian, spanish and french right now


For more details and examples visit the plugin page on <a href="http://blog.mypuzzle.org/sudoku-for-wordpress/">Wordpress sudoku Plugin</a>

== Installation ==

1. Upload 'Sudoku Mypuzzle' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the MyPuzzle Sudoku Options page under settings and save your preferred options.
4. Use the Sudoku code "[sudoku-mp]" to insert sudoku puzzles into your posts/pages.

The <a href="http://mypuzzle.org/sudoku">sudoku</a> are provided by mypuzzle.org.

== How to use ==

Use the wordpress post shortcut [sudoku-mp] for default setup or with parameter for individualizing for each post.
Examples:
- [sudoku-mp]
- [sudoku-mp size=1 level=1 showlevel=1 showhistory=1 showprint=0 showtimer=0 showlink=1]

Visit <a href="http://blog.mypuzzle.org/sudoku-for-wordpress/">Wordpress sudoku Plugin</a>
